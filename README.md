### This Project is in it's early stage

## How to install
To install dependencies use : `pip install -r requirements.txt`

## Needed
This script needs both deezer api and Google Oauth
## Deezer auth
* Go to Deezer developpers : https://developers.deezer.com/api
* My apps (on top right)
* Create
* Select a name, a domain, redirect url, link to terms of use (all of that can be anything)
* Enter your app ID and secret key inside edit_me.py

## Google Oauth
* Go to google console
* Create a project
* Click on APIs & auth -> APIs
* Enable YoutubeApiV3
* Create a client ID
* Download JSON
* Use the json in your project folder and change client_secrets_file with the name of your file

## Run
Run, run_me.py and follow the instructions(choose 1 for liked playlist and 2 for a selected playlist)
